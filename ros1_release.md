# ROS1 release

ROS1 module is implemented with Noetic distribution to localize pointed targets on a planar workspace surface.
The module utilizes [OpenPose](https://arxiv.org/abs/1812.08008)-based pose estimation node developed in [OpenDR project](https://github.com/opendr-eu/opendr).
